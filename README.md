# The-adventures
## Installation / running
**Linux / Windows**
In order to run this game you need to download latest version from releases section. Then go to the directory of the downloaded .jar file and run command `java -jar filename.jar` (Replace 'filename' with the actual name of the file). Then you should be good to go.

**Android**
There is no way to natively run java code on android so you would need to use some kind of emulator or other application for running java code. One of the options is Linux emulator Termux where you can just copy or download the .jar file and run it like on PC.
